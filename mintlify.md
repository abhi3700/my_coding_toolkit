# Mintlify

## Description

Doc creation tool for docs.

## Editor

VSCode extensions

- [Mintlify Doc Writer](https://marketplace.visualstudio.com/items?itemName=mintlify.document)
- [Mintlify MDX](https://marketplace.visualstudio.com/items?itemName=mintlify.mintlify-snippets)
- [MDX](https://marketplace.visualstudio.com/items?itemName=unifiedjs.vscode-mdx)

## CLI

```sh
npm i -g mint
```

## Getting Started

Use this [Starter Kit](https://github.com/mintlify/starter) as template for your own doc.
